In this article and the next ones, we'll dive into all the methods that you can use with arrays in JavaScript. And interactively, so you can understand them quickly and forever!


Today we'll have a look at the map and forEach methods, and the filter method.





Why array methods and where to find them:



I already commented on these topics in the first part of this series. You can read it here: Learn Javascript Array Methods Interactively - Part 1.


So let's start:



Map() & forEach():



Both had the following optional parameters:


- element: This argument represents the current element being processed in the array.

- index: This argument represents the index of the current element being processed in the array.

- array: It is the original array on which map() is being applied.


Both the map() and forEach() methods are used to iterate over elements in an array and apply a provided function to each element. However, see the difference between the two methods:



MAP(): Create a new array by applying a provided function to each element of the original array:



let originalArray = [ ];


let newArray = [ ];



originalArray.map((element) => element * );





FOREACH(): Perform actions on each element of an array (but doesn't create a new array):



In this example, we're going to do the same thing as we did with the map method in the first example:



let originalArray = [ ];


let newArray = [ ];



originalArray.forEach((num) => { result.push(num * });





As you can see, we cannot make it as simple as with the map method, because the forEach method does not return anything. So, we have to push the result in a new array we have initialized ourself beforehand.


We had a look at the Push() in part 1 of this series. You can read it: Learn Javascript Array Methods Interactively - Part 1.



More MAP() examples, using an external function:

function capitalize(word) {
  return word.charAt(0).toUpperCase() + word.slice(1);
}



function getLength(name) {
  return name.length;
}





To do the same with forEach(), once more, we would have to push the results into another array. We would initialize a new array: capitalizedNames = [ ] and then we would push the results into it: originalArray.forEach((name) => { capitalizedNames.push (capitalize(name) )});



Filter():



The filter() method in JavaScript is used to create a new array containing elements from an existing array that satisfy a certain condition.

It iterates over each element of the array and applies a callback function to determine whether to include the element in the filtered array.
