Today we'll have a look at the pop, push, shift, unshift, splice, and slice methods.





Why array methods?



Arrays are an essential data structure in JavaScript, providing a way to store and manipulate collections of values.



JavaScript provides a rich set of built-in array methods that empower developers to perform various operations efficiently. Understanding and utilizing these methods can greatly simplify your code and make it more expressive.



Where can you find them?



If you want to see all the array methods in JavaScript, you can log the Array.prototype object in the console:



👉 The Array.prototype object in JavaScript is the prototype object for all arrays. It contains properties and methods that are inherited by all array instances created in JavaScript.



When you create an array using the array literal syntax ([ ]) or the Array constructor, the newly created array object inherits properties and methods from the Array.prototype object. This means that you can access and use those properties and methods on any array instance.

3. Create a new array that contains a shallow copy of a portion of an existing array.

It does not modify the original array.



The slice() method takes two optional parameters: start and end.

The start parameter specifies the index at which to start extracting elements (inclusive), and the end parameter specifies the index at which to stop extracting elements (exclusive).

If no end parameter is provided, slice() extracts elements from the start index until the end of the array.

4. Change the contents of an array by removing, replacing, or adding elements in place.

It modifies the original array and returns a new array containing the removed elements.



The splice() method takes in multiple parameters:

- start: The index at which to start modifying the array. If there is only one parameter, the splice method will remove all elements from that start index to the end of the array.

- deleteCount (optional): The number of elements to remove starting from the start index. If set to 0, no elements will be removed.

- item1 , item2 , ... (optional): Elements to add to the original array at the start index.
