In this article we're going to see what is the Record Utility Type and some examples to understand it, including a mini grades app.




What is the Record Utility Type?



The Record Utility Type is used to create a new type with specified keys mapped to a specific value type. It takes two type parameters: K for the keys and V for the values.



Here you can see a very easy example:





In this example, 'NumberDictionary' is a type that represents an object with string keys and number values.



However, the Record utility type can hold any type for both the keys and the values. This includes type aliases and literal types, such as string literals, number literals, boolean literals, and even symbol literals.
